1. What is the difference between var, let, and const?

Ans : Difference between var, let, and const

    i. var: Function-scoped, can be redeclared and updated.
    ii. let: Block-scoped, can be updated but cannot be redeclared in the same scope.
    iii. const: Block-scoped, cannot be redeclared or updated. For objects/arrays, the reference cannot change, but properties/elements can be modified.

2. What is the spread operator (...)?

Ans : 
    
    Makes copies of stuff, or spreads things around, depending on mood. Works on arrays, objects, and sometimes other things.

3. What is the difference between map(), filter(), and forEach()?

Ans : Difference between map(), filter(), and forEach()

    i. map() – makes a new array from old array, kind of like magic.
    ii. filter() – picks only some items, maybe the good ones.
    iii. forEach() – goes through items, does things, but doesn’t return anything you can use later.

4. What is an arrow function?

Ans :

    Shorter function style with arrows. Sometimes it behaves like a normal function, sometimes it doesn’t.

5. What are template literals?

Ans :

    Strings with backticks. Lets you put ${something} inside without using plus signs, kinda handy but weird sometimes.
